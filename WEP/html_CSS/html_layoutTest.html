벤치프레스 운동방법

2020년 06월 10일

시작 자세

1. 바벨이 턱 바로 위에 위치하도록 벤치에 눕는다.

2. 가슴을 최대한 펴준 상태로 날개뼈를 벤치 안으로 모아준다고 상상하면서 바벨을 어깨보다 조금 넓게 잡아준다.


3. 이때 발은 엉덩이 쪽으로 당겨준 다음,

바닥에 단단히 고정시킨다.

운동법

1. 등이 최대한 젖혀지게 벤치에 누운다음,

(허리가 아닌 등을 젖힐 것).


2. 바벨을 구부려 준다고 상상하며,

당기는 느낌으로 내려준다."

(숨을 들이쉬며)


3.

숨을 내쉬면서,

가슴을 내밀어준다는 느낌에 최대한 집중하면서 바벨을 밀어준다.

4. 다시 숨을 들이쉬면서, 바벨을 당기는 느낌으로 내려주는 걸 반복한다.
